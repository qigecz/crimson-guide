## 最佳前期武器

在《红色沙漠》(Crimson Desert) 中，尽早获得一把强力武器会让你受益匪浅。以下是你在前15小时内应该着手获取的武器。

---

## 顶级前期武器

### 1. 花郎刀 (Hwando) — 最佳综合前期武器

| 属性 | 数值 |
|------|------|
| **攻击力** | 高 |
| **速度** | 中快 |
| **属性成长** | STR / DEX |
| **获取方式** | 在赫南德 (Hernand) 地区的宝箱中获得 |
| **独特之处** | 可注入深渊元素 |

**花郎刀 (Hwando)** 被公认为最佳前期武器，原因如下：
- 获取时间很早（第1-2章）
- 基础伤害高
- 支持深渊神器升级
- 招式体系适合大多数敌人类型

### 2. 持续供给箭 (Sustained Supply Arrow) — 最佳远程武器

| 属性 | 数值 |
|------|------|
| **类型** | 无限箭矢 |
| **伤害** | 中等 |
| **获取方式** | 与波洛林村 (Pororin Village) 的卢米尔 (Lumiel) 对话 |
| **独特之处** | 永不耗尽弹药 |

这本质上是一个**无限箭矢供给**。在波洛林村 (Pororin Village) 与**研究主管**对话获取。它是"森林守护者"任务线的奖励。

> **优先级：** 尽快获取。无限箭矢意味着你可以随意使用远程攻击，不必担心资源消耗。

### 3. 采矿拳钻 (Mining Knuckledrill) — 功能性武器

| 属性 | 数值 |
|------|------|
| **类型** | 采矿工具 / 武器 |
| **获取方式** | 击败马尼的挖掘机 (Marni's Excavatron) 或流血强盗 (Bleed Bandits) |
| **独特之处** | 采集矿脉的必备工具 |

这不是一把战斗武器，但对游戏进程至关重要。你需要它来采集矿石，用于武器和护甲升级。

## 锻造（升级）系统

武器可以在任何**铁匠铺**使用采集到的材料进行升级。

### 如何锻造

1. 前往主要聚落中的铁匠铺
2. 选择要升级的武器
3. 支付材料费用 + 银币
4. 每级锻造提升攻击力

### 所需材料

| 材料 | 来源 | 用途 |
|------|------|------|
| 铁矿石 | 矿脉（需要拳钻） | +1 至 +3 |
| 钢锭 | 铁匠熔炼（2个铁矿石） | +4 至 +6 |
| 月石 | Boss掉落、古代遗迹 | +7 至 +9 |
| 龙鳞 | 仅后期Boss掉落 | +10 以上 |

### 锻造技巧

- **始终优先升级主武器** — 伤害提升最为关键
- **前期不要分散升级** 多把武器
- 你可以在附魔台更换**元素附魔**
- 部分武器拥有通过特殊任务解锁的**隐藏升级路线**

## 深渊神器系统

深渊神器是提供被动加成的特殊物品。可以装备它们来强化你的Build。

### 如何获取深渊神器

1. **深渊神器 (Abyss Artifacts)** — 在副本和隐藏区域中获得
2. **深渊枢纽 (Abyss Nexus)** — 与之互动以解锁加成
3. **Boss掉落** — 部分Boss掉落独特的深渊物品
4. **任务奖励** — 特定任务线奖励深渊神器

### 最佳前期深渊神器

| 神器 | 效果 | 来源 |
|------|------|------|
| 公理之力 (Axiom Force) | 战斗牵引、拉近敌人 | 技能解锁 |
| 深渊核心 - 攻击 | +5% 攻击力 | 圣殿战利品 |
| 深渊核心 - 防御 | +5% 防御力 | 圣殿战利品 |
| 深渊核心 - 耐力 | +10% 耐力 | 古代遗迹 |

### 如何装备深渊核心

1. 从暂停菜单打开**深渊菜单**
2. 进入**核心配置**
3. 将核心拖入可用槽位
4. 每个核心提供一个被动加成
5. 可以随时更换核心（无消耗）

> **提示：** 前期优先选择**耐力核心** — 它们提供最大的体验改善。
