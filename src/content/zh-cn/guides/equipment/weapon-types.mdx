## 武器类型与升级指南

理解武器系统是掌握《红色沙漠》(Crimson Desert) 战斗的关键。与许多将武器视为简单属性道具的动作RPG不同，《红色沙漠》中的每种武器类型都拥有**独特的招式体系**，从根本上改变了你的战斗方式。选择适合自己游戏风格的武器，并持续进行升级——这将决定你是在关卡中苦苦挣扎，还是轻松制霸。

本指南将详细讲解每种武器类别、锻造（升级）系统、元素附魔以及隐藏升级路线。

|------|------|------|------|----------|-----------|

| **双手剑** | ★★☆ | ★★★★★ | ★★★ | 中等 | 缓慢、重甲敌人 |

| **盾牌**（攻击型） | ★★★ | ★★★ | ★★ | 中等 | 防御型玩法、破防 |

## 单手剑

最平衡、最万能的武器类型。单手剑可以搭配盾牌使用，连招可靠稳定。

### 招式详解

| 攻击 | 输入 | 属性 |
|------|------|------|
| **斩击连招** | 轻击 x4 | 4段连击，出招快 |
| **上挑斩** | 重击（短按） | 向上斩击，可挑空轻型敌人 |
| **蓄力突刺** | 重击（长按） | 远距离刺击，高体势伤害 |
| **旋刃** | 轻击 x3 + 重击 | 连招终结技，大范围横扫 |
| **盾击连招** | 轻击 + 盾击 | 混合攻击，打断敌人节奏 |

### 属性成长

- **主属性**：力量 (STR)
- **副属性**：敏捷 (DEX)
- **最佳前期武器**：花郎刀 (Hwando)（在赫南德 (Hernand) 地区的宝箱中获得）

### 何时使用单手剑

- 你想要一种**平衡的战斗风格**，能适应各种情况
- 你在面对**混合敌人群体**，需要兼顾攻防
- 你依赖**盾击连招**和盾技技能树
- 你是游戏新手，想要最易上手的武器类型

> **提示：** 花郎刀是前15小时以上最佳的单手剑。它的招式流畅，且支持深渊神器升级。获取方法请参考我们的[最佳前期武器指南](/zh-cn/guides/equipment/best-early-weapons)。

## 双手剑

牺牲速度换取纯粹破坏力的巨型利刃。双手剑擅长破坏敌人体势，对付重甲敌人效果拔群。

### 招式详解

| 攻击 | 输入 | 属性 |
|------|------|------|
| **劈砍连招** | 轻击 x3 | 3段连击，横扫范围大 |
| **过顶重砸** | 重击（短按） | 纵向下砸，极高破防效果 |
| **蓄力劈砍** | 重击（长按） | 蓄力大范围横扫，可破防 |
| **裂地击** | 轻击 x2 + 重击（长按） | 范围终结技，地面重击 |
| **旋风斩** | 重击 + 闪避 | 旋转后退攻击 |

### 属性成长

- **主属性**：力量 (STR)
- **副属性**：活力 (VIT)
- **最佳前期武器**：铁巨剑 (Iron Greatsword)（在赫南德 (Hernand) 的铁匠铺购买）

### 何时使用双手剑

- 你在对付**重甲敌人**，轻武器难以造成伤害
- 你想要**快速破坏Boss体势**
- 你偏好**单次高伤害**，而非持续连击
- 你习惯于**较慢的节奏**和蓄意攻击

> **重要提示：** 双手剑每次挥击消耗的耐力显著更多。请投资耐力升级和耐力深渊核心，以维持攻击压力。

## 双刀

游戏中攻击速度最快的武器类型。双刀以迅捷攻击压制敌人，能以惊人的速度叠加异常状态和连击。

### 招式详解

| 攻击 | 输入 | 属性 |
|------|------|------|
| **疾风连斩** | 轻击 x5 | 5段快速连击 |
| **十字斩** | 重击（短按） | X形双刀交叉斩 |
| **旋舞** | 轻击 x4 + 重击 | 多段旋转终结技 |
| **影步** | 闪避 + 轻击 | 闪避动作中攻击 |
| **空中裂斩** | 跳跃 + 轻击 x3 | 空中连击，对付飞行敌人效果好 |

### 属性成长

- **主属性**：敏捷 (DEX)
- **副属性**：力量 (STR)
- **最佳前期武器**：双獠牙 (Twin Fangs)（卡林采石场 (Karin Quarry) 的流血强盗掉落）

### 何时使用双刀

- 你想要**游戏中最快的攻击速度**
- 你在对付**敏捷、善闪避的敌人**，慢速武器难以命中
- 你喜欢**高操作频率**的战斗体验
- 你想要**快速叠加异常状态**（中毒、流血、灼烧）

> **注意：** 双刀的攻击距离很短，单次伤害也较低。你需要在危险范围内停留更长时间，因此闪避的耐力管理至关重要。

## 长柄武器：长枪与戟

### 长枪

长枪拥有游戏中最长的近战距离，让你能在安全位置 poke 敌人。

| 攻击 | 输入 | 属性 |
|------|------|------|
| **突刺连招** | 轻击 x4 | 快速连刺，距离远 |
| **横扫** | 重击（短按） | 大范围水平挥击 |
| **冲刺突刺** | 重击（长按） | 远距离冲刺刺击 |
| **风车刺** | 轻击 x3 + 重击 | 多方向旋转突刺 |

**属性成长**：敏捷（主属性）、力量（副属性）

**最适合**：与危险敌人保持距离、在走廊中战斗、用横扫攻击进行群体控制。

### 戟

结合突刺和横扫攻击的混合型武器。最全能但也最难精通的近战武器。

| 攻击 | 输入 | 属性 |
|------|------|------|
| **劈砍连招** | 轻击 x3 | 突刺与横扫交替 |
| **撑杆跳** | 重击（短按） | 越过敌人，背后攻击 |
| **旋风** | 重击（长按） | 旋转多段攻击 |
| **长驱直入** | 轻击 x2 + 重击 | 冲刺突刺终结技 |

**属性成长**：力量（主属性）、敏捷（副属性）

**最适合**：喜欢多面手武器的玩家。戟可以像长枪一样突刺，也可以像巨剑一样横扫，是最灵活多变的近战武器。

## 远程武器

### 弓

弓提供无声远程攻击，可使用多种箭矢。非常适合引怪和处理飞行敌人。

| 箭矢类型 | 效果 | 来源 |
|----------|------|------|
| **普通箭** | 常规伤害 | 购买 / 制作 |
| **火箭** | 火焰伤害 + 持续伤害 | 用火焰精华制作 |
| **毒箭** | 中毒持续伤害 | 用毒腺制作 |
| **穿甲箭** | 无视护甲 | 稀有掉落、铁匠铺 |
| **牵引箭** | 将敌人拉向你 | 技能解锁 |

**属性成长**：敏捷（主属性）

**最佳前期弓**：持续供给箭 (Sustained Supply Arrow)（无限箭矢——波洛林村 (Pororin Village) 任务奖励）

> **优先级：** 尽早获取持续供给箭。无限箭矢彻底消除了远程战斗的资源管理压力。

### 弩

单发伤害高于弓，但装填速度较慢。弩可以单手使用，允许搭配盾牌。

| 类型 | 装填速度 | 伤害 | 特殊效果 |
|------|----------|------|----------|
| **轻型弩** | 快 | 中等 | 装填时可移动 |
| **重型弩** | 慢 | 高 | 穿透多个敌人 |
| **连弩** | 很快 | 单发较低 | 一次齐射发射3支弩箭 |

## 锻造（升级）系统

武器可以在主要聚落中的**铁匠铺**进行升级。锻造可以提升武器的基础攻击力。

### 如何锻造

1. 前往主要聚落中的铁匠铺
2. 选择要升级的武器
3. 支付材料费用 + 银币
4. 每级锻造提升基础攻击力
5. 在特定阈值（+5、+7、+10）时，会附加额外属性加成

### 各等级所需材料

| 锻造等级 | 所需材料 | 银币费用 | 来源 |
|----------|----------|----------|------|
| **+1 至 +3** | 铁矿石 x2（每级） | 每个 50 银币 | 矿脉（需要拳钻 (Knuckledrill)） |
| **+4 至 +6** | 钢锭 x2（每级） | 每个 150 银币 | 铁匠熔炼（2个铁矿石 = 1个钢锭） |
| **+7 至 +9** | 月石 x1（每级） | 每个 500 银币 | Boss掉落、古代遗迹、圣殿 |
| **+10 以上** | 龙鳞 x1（每级） | 每个 2000 银币 | 仅后期Boss掉落 |

### 锻造加成阈值

| 等级 | 加成 |
|------|------|
| **+5** | +10% 暴击率 |
| **+7** | +5% 元素伤害 |
| **+10** | 武器独特加成（因武器而异） |

> **提示：** 始终**优先升级主武器**。前期不要将材料分散投资到多把武器上。一把+7的武器远胜于三把+3的武器。

## 元素附魔

在**附魔台**（位于主要聚落铁匠铺附近）可以为武器注入元素属性。

### 可用元素

| 元素 | 效果 | 最适合对付 | 附魔材料 |
|------|------|-----------|----------|
| **火** | +20% 伤害，灼烧持续伤害 | 冰属性敌人、沙漠守卫 (Desert Guardian) | 火焰精华（火焰敌人掉落） |
| **冰** | +15% 伤害，减速敌人 | 快速敌人、野兽 | 霜碎片（冰属性敌人掉落） |
| **雷** | +15% 伤害，连锁附近敌人 | 敌人群组 | 风暴水晶（稀有掉落） |
| **深渊** | +25% 对深渊敌人伤害 | 深渊类敌人、终局内容 | 深渊碎片（深渊副本掉落） |
| **神圣** | +20% 对亡灵伤害 | 亡灵、幽灵类敌人 | 圣水（教堂任务奖励） |

### 更换附魔

你可以在任何附魔台**更换武器的元素附魔**。更换需要消耗银币和对应的元素材料。之前的附魔会被覆盖。

> **策略建议：** 携带多种元素材料，以便在Boss战前切换附魔。打沙漠守卫用火，打亡灵副本用神圣，以此类推。

## 隐藏升级路线

《红色沙漠》中的部分武器拥有**隐藏升级路线**，不会在标准锻造菜单中显示。这些路线需要完成特定任务或找到特殊NPC才能解锁。

### 已知隐藏路线

| 武器 | 触发条件 | 结果 |
|------|----------|------|
| **花郎刀 (Hwando)** | 完成"灰鬃遗产"任务线 | 花郎刀转化为**深渊花郎刀** — 独特的深渊招式 |
| **铁巨剑 (Iron Greatsword)** | 在嚎叫丘陵 (Howling Hill) 找到古代铁匠 | 解锁**月铸巨剑**路线 |
| **双獠牙 (Twin Fangs)** | 用流血击杀击败100个敌人 | 进化为**猩红獠牙** — 增强流血伤害 |
| **采矿拳钻 (Mining Knuckledrill)** | 采集50个矿脉 | 升级为**精金钻** — 更快采矿，可采集月石 |

## 武器与技能协同

某些技能与特定武器类型配合更佳。请据此规划你的Build：

| 技能树 | 最佳武器搭配 | 原因 |
|--------|-------------|------|
| **剑术精通** | 单手剑 / 双刀 | 剑技的专属技能树 |
| **盾技** | 单手剑 + 盾牌 | 盾牌战斗需要单手武器 |
| **摔跤** | 任意（以抓投为主） | 投技与武器类型无关 |
| **箭术** | 弓 / 弩 | 弓箭专属技能和射击技巧 |
| **敏锐感知** | 所有武器 | 闪避/弹反通用，所有Build受益 |

## 升级优先级建议

在游戏前20小时，请遵循以下升级优先级：

1. **主武器升到+5** — 单项最大伤害提升
2. **主武器元素附魔** — 匹配当前区域的敌人属性
3. **采矿拳钻 (Mining Knuckledrill)** — 解锁矿石采集，为后续升级做准备
4. **主武器升到+7** — 获得+5暴击加成后的显著战力提升
5. **副武器升到+3** — 元素多样性的备用方案
6. **主武器升到+10** — 如果你从后期Boss处获得了龙鳞

> **最终建议：** 不必纠结于选"错"了武器。《红色沙漠》的设计确保每种武器类型在整个游戏流程中都可行。选择你觉得顺手的武器，全力投资，深入掌握其招式。一把精通的武器永远胜过一把未精通的"更好"的武器。
