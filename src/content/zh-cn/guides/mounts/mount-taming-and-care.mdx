## 坐骑系统概述

坐骑在《红色沙漠》中至关重要。皮威尔大陆幅员辽阔，步行到任何地方都不现实。尽早掌握坐骑系统将为你节省大量旅行时间。

---

## 如何驯马

### 基本驯马流程

1. **找到野马** — 在特定的草原和平原区域刷新
2. **缓慢接近** — 使用蹲伏避免惊吓
3. **使用信任系统** — 喂食苹果或干草建立信任
4. **上马** — 信任度足够高时尝试骑乘
5. **完成小游戏** — 保持平衡不被甩下

### 驯马技巧

| 技巧 | 说明 |
|------|------|
| 从后方接近 | 马有视觉盲区——利用它 |
| 使用高级饵料 | 苹果有效，但方糖效果更好 |
| 收起武器 | 鞘中武器减少敌意范围 |
| 时间段很重要 | 黎明和黄昏野马刷新更多 |
| 保持耐心 | 急于求成会重置信任度 |

### 信任系统

每匹马都有**信任等级**，决定了：

- 能否骑乘
- 执行命令的速度
- 最大速度和体力
- 跨越危险地形的意愿

**提升信任的方法：**
- 定期喂食（苹果、干草、方糖）
- 不要让马的体力耗尽
- 在营地为马梳理毛发
- 骑马参与战斗（提升战斗信任）

## 马匹属性

每匹坐骑有四个核心属性：

| 属性 | 效果 |
|------|------|
| **速度** | 最大移动速度 |
| **体力** | 冲刺持续时间 |
| **胆量** | 被敌人惊吓的抗性 |
| **负重** | 骑乘时的背包容量 |

## 治疗你的马

马匹会因战斗、坠落和环境危害而受伤。

### 治疗方法

1. **马匹生命药水** — 在马厩购买或自行制作
2. **营地休息** — 缓慢恢复
3. **喂食苹果** — 少量恢复生命
4. **前往马厩** — 完全治愈和护理

> **警告：** 如果马的生命值降到 0，它会**暂时无法使用**，直到你在马厩治疗。进入危险区域前务必做好准备。

## 传奇坐骑

《红色沙漠》中最受追捧的坐骑。它们拥有独特的外观和远超普通马匹的属性。

### 已知传奇坐骑

| 坐骑 | 位置 | 获取方式 |
|------|------|----------|
| **白色风暴** | 北方山峰 | 完成「潜伏之狼」任务链 |
| **沙漠幽影** | 皮威尔荒原 | 野外驯服（极稀有刷新） |
| **铁甲** | 第七章奖励 | 剧情保证获得 |
| **深渊奔行者** | 隐藏深渊地牢 | 解开「过境点」谜题 |

> **白色风暴**被广泛认为是综合最佳坐骑，因为它拥有均衡的属性和独特的体力恢复能力。

## 马厩管理

大多数主要聚落都有马厩，提供：

- **寄存** — 存放额外坐骑
- **治疗** — 恢复马匹生命
- **梳理** — 提升信任和外观
- **装备** — 购买马鞍、马铠和鞍袋
- **交易** — 出售或交换坐骑（部分NPC提供特殊交易）

### 马匹装备

| 部位 | 效果 |
|------|------|
| 马鞍 | 提升体力恢复 |
| 马铠 | 增加骑乘战斗防御 |
| 鞍袋 | 增加负重 |
| 缰绳 | 提升转向速度和控制力 |

## 骑马战斗技巧

1. **打完就跑** — 利用速度优势，不要贪刀
2. **瞄准眩晕** — 骑乘攻击额外增加击退效果
3. **知道何时下马** — 部分BOSS需要步战
4. **保持退路** — 骑马时切勿被围堵
5. **备好马匹药水** — 探索危险区域前至少携带 3 瓶
