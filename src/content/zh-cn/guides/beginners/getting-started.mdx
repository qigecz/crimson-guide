## 欢迎来到红色沙漠

本攻略将涵盖你在进入红色沙漠世界之前需要了解的基本知识。

### 操作与设置

- **移动**：WASD 键 / 左摇杆
- **攻击**：左键 / RT
- **闪避**：空格 / B 键
- **互动**：E 键 / A 键

### 第一步

刚开始游戏时，你会出现在一个简短的教程区域。请仔细学习战斗机制——这将在整个旅程中至关重要。

### 战斗基础

红色沙漠拥有深度的动作战斗系统。尽早掌握这些基本操作：

1. **轻攻击** 速度快但伤害较低
2. **重攻击** 速度较慢但可以击退敌人
3. **闪避** 有无敌帧——掌握好时机
4. **格挡** 可以让敌人露出破绽，进行致命一击

### 新手建议

- 完成教程关卡
- 仔细探索起始区域的隐藏物品
- 与 NPC 对话获取支线任务
- 在第一个铁匠处升级你的武器
