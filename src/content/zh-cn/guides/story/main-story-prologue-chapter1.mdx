## 序章：深夜（Dead of Night）

《红色沙漠》的故事从 **克利夫**（Kliff）——灰鬃雇佣兵团的团长开始。你的营地突然遭到黑熊军团（Black Bears）的袭击，由此拉开了横跨皮维尔大陆的史诗冒险。

序章不仅是一段剧情引入，更是一个精心设计的战斗和系统教程。请务必认真学习每一个机制——它们将在整个旅途中反复考验你。

### 序章剧情背景

灰鬃雇佣兵团是皮维尔大陆上众多佣兵组织之一。克利夫作为团长，带领团员们接取各种委托维持生计。在游戏开始时，灰鬃团的营地驻扎在赫南德地区外围，一切看似平静——直到黑熊军团突然发起夜袭。

**关键角色：**

| 角色 | 身份 | 说明 |
|------|------|------|
| **克利夫** | 灰鬃团团长 | 主角，冷静沉着的战士 |
| **黑熊团长** | 黑熊军团首领 | 序章的幕后指挥者 |
| **灰鬃团员们** | 克利夫的战友 | 教程中的引导NPC |

### 伏击战（Ambush）

#### 战斗教程阶段

开场既是剧情引入，也是战斗教程。你将分阶段学习到：

**第一阶段：基础攻击**
- **轻攻击** — 快速挥砍，连按可形成三连击
- **重攻击** — 蓄力重劈，可击退前方敌人
- **组合攻击** — 轻攻击和重攻击自由组合

**第二阶段：防御系统**
- **格挡** — 按住格挡键减少受到的伤害
- **弹反（完美格挡）** — 在敌人攻击命中的瞬间按下格挡，可弹开攻击并让敌人露出破绽
- **弹反是整个战斗系统的核心** — 后续几乎所有BOSS战都需要精通弹反

**第三阶段：闪避与移动**
- **闪避** — 通过无敌帧规避伤害
- **闪避方向** — 可以向任何方向闪避
- **冲刺闪避** — 冲刺中的闪避距离更远

> **提示：** 认真学习弹反教程。这个机制是《红色沙漠》战斗系统的核心，将在整个游戏中反复考验你。如果你在教程中未能成功弹反，游戏会反复提示直到你掌握——请不要跳过这个环节。

#### 伏击战BOSS：黑熊先锋

| 信息 | 详情 |
|------|------|
| **等级** | 3 |
| **攻击类型** | 重击为主 |
| **弱点** | 攻击后有大硬直 |
| **推荐策略** | 弹反反击 |

**攻击模式：**

| 攻击 | 前摇 | 应对方式 |
|------|------|----------|
| 横扫劈 | 举手蓄力0.5秒 | 弹反或后退闪避 |
| 重劈 | 双手举过头顶1秒 | 弹反（最佳时机） |
| 冲撞 | 后退一步后前冲 | 侧闪+反击 |
| 连续三击 | 先轻后重 | 第一击格挡，第三击弹反 |

> **战术要点：** 黑熊先锋的所有攻击都有明显的蓄力前摇。不要贪攻——等待他的重劈，弹反后接一套轻攻击连招。保持耐心，3-4次弹反反击即可击杀。

### 未知空间（Unknown Space）

伏击之后，克利夫被拉入一个神秘维度。这一段将介绍游戏的核心超自然设定。

#### 探索要点

- **深渊（Abyss）** — 游戏核心设定的超自然力量，贯穿整个故事
- 基础**探索机制** — 使用灯笼寻找隐藏路径和物品
- 首次接触**深渊神器** — 可提供属性加成的收集品
- **深渊能量** — 使用深渊技能的能量来源

#### 收集品清单

| 物品 | 位置 | 效果 |
|------|------|------|
| **深渊碎片（小）** | 入口左侧通道 | 深渊能量+1 |
| **古老卷轴** | 主路尽头高台 | 解锁背景故事 |
| **深渊结晶** | 隐藏通道末端 | 技能点+1 |
| **神秘项链** | 暗室角落（灯笼照亮） | 深渊共鸣+5% |

> **重要：** 未知空间中的所有物品一旦离开后无法再获取。仔细探索每一个角落，使用灯笼照亮暗处。

---

### 序章注意事项

1. **离开营地前与所有灰鬃成员对话** — 部分对话会解锁后续支线任务，每个团员都有独特的对话内容
2. **在逃亡过程中收集所有战利品** — 序章的战利品包含稀有材料，离开后无法返回
3. **不要跳过战斗教程** — 后续章节的难度会显著提升，没有扎实的基本功会很吃力
4. **在未知空间中充分探索** — 这里的深渊碎片和结晶是免费的属性提升
5. **注意剧情细节** — 序章中的对话和场景暗示了许多后续剧情线索

### 主线任务一：善良的考验（Trials of Kindness）

第一个主线任务将你介绍给当地居民和这片动荡的区域。

#### 任务目标

**阶段一：抵达赫南德镇**
- 跟随路径指引到达赫南德镇
- 沿途会遇到零散敌人（2-3级），是练手的好机会
- 激活路径上的所有传送点

**阶段二：帮助村民**
- 帮助铁匠搬运材料（教学：学习锻造系统）
- 帮助农场主驱赶野兽（教学：学习狩猎系统）
- 帮助商人护送货物的路线（教学：学习护送任务机制）

**阶段三：贡献点数系统**
- 学习**贡献点数**系统——通过帮助当地人获取贡献点数
- 贡献点数可以兑换装备、解锁营火功能、获得NPC折扣
- 完成所有可选目标可以让你在赫南德的贡献奖励中获得先发优势

#### 任务奖励

| 奖励 | 条件 |
|------|------|
| **500 经验** | 完成主线目标 |
| **赫南德之友**称号 | 完成所有可选目标 |
| **精良护腕** | 完成所有可选目标额外奖励 |
| **铁匠好感度+1** | 帮助铁匠搬运 |
| **商人好感度+1** | 帮助商人护送 |

> **提示：** 完成所有可选目标，可以让你在赫南德的贡献奖励中获得先发优势，包含珍贵的护甲部件。铁匠好感度影响锻造价格，商人好感度影响商品价格——两个都值得投资。

### 主线任务二：痕迹（Traces）

这个任务开始调查黑熊军团和神秘势力的线索。

#### 任务流程

**阶段一：线索收集**
- 在赫南德镇外的营地找到黑熊军团的痕迹
- 使用追踪技能追踪脚印和线索
- 发现黑熊军团正在搜索某件「深渊遗物」

**阶段二：潜入调查**
- 潜入黑熊军团的前哨站
- 可以选择**正面突破**或**潜行渗透**
- 潜行路线可以获得额外情报和战利品
- 正面突破可以获得更多战斗经验

**阶段三：真相浮现**
- 发现黑熊军团正在寻找的「深渊遗物」
- 遇到神秘NPC「旅人」——暗示后续剧情走向
- 触发第一章BOSS战的前置事件

#### 任务奖励

| 奖励 | 条件 |
|------|------|
| **800 经验** | 完成主线 |
| **深渊碎片 x3** | 完成主线 |
| **黑熊军团情报** | 潜行路线额外奖励 |
| **稀有材料包** | 完成所有可选目标 |

### 可接支线任务详解

#### 支线一：两极对立（Polar Opposites）

**任务说明：** 赫南德镇的两位炼金术士正在进行一场关于元素理论的争论，你需要帮助他们验证各自的学说。

| 信息 | 详情 |
|------|------|
| **推荐等级** | 3+ |
| **任务类型** | 收集+战斗 |
| **奖励** | 元素之心（饰品）、300经验 |

**任务流程：**
1. 接取任务——在赫南德镇找到两位炼金术士
2. 前往冰沼收集「冰霜精华」
3. 前往矿脉山谷收集「火焰结晶」
4. 将两种材料分别交给两位炼金术士
5. 协助他们完成元素融合实验（解谜小游戏）
6. 获得元素之心饰品（元素伤害+5%）

#### 支线二：失衡深渊（Abyss Without Balance）

**任务说明：** 赫南德附近出现了一处不稳定的深渊裂隙，需要你去调查和稳定它。

| 信息 | 详情 |
|------|------|
| **推荐等级** | 5+ |
| **任务类型** | 解谜+战斗 |
| **奖励** | 深渊共鸣+3%、500经验、深渊碎片 x5 |

**任务流程：**
1. 前往裂隙位置（赫南德镇北方山谷）
2. 使用灯笼探测深渊能量流向
3. 按正确顺序激活三个深渊锚点（谜题）
4. 击败从裂隙中涌出的深渊生物（5级精英x2）
5. 使用深渊能力封印裂隙
6. 获得深渊共鸣永久提升

> **深渊谜题解法：** 观察三个锚点上的符文颜色——按照从浅到深的顺序激活即可。如果激活错误顺序，会出现额外的深渊怪物。

#### 支线三：暗流涌动（Undercurrents）

**任务说明：** 赫南德镇的银行经理行为异常，似乎与某个地下组织有联系。

| 信息 | 详情 |
|------|------|
| **推荐等级** | 4+ |
| **任务类型** | 潜行+调查 |
| **奖励** | 银行钥匙、400经验、面具 |

这个支线与**赫南德银行劫案**直接相关。完成此任务可以获得银行钥匙，是进行银行劫案的两种途径之一。

### 第一章BOSS：黑熊精锐

| 信息 | 详情 |
|------|------|
| **等级** | 8 |
| **生命值** | ★★★☆☆ |
| **攻击力** | ★★★☆☆ |
| **速度** | ★★☆☆☆ |
| **架势条** | ★★★☆☆ |
| **弱点** | 头部和背部 |

**攻击模式与应对：**

| 攻击 | 伤害 | 前摇 | 应对 |
|------|------|------|------|
| 三连斩 | 中 | 短 | 第三击弹反 |
| 盾击 | 低+眩晕 | 中 | 闪避 |
| 冲锋劈 | 高 | 长（后退一步） | 弹反最佳时机 |
| 呼唤援兵 | 无 | 长 | 打断或快速击杀援兵 |
| 激怒模式 | 自身攻击+30% | 触发条件：生命低于50% | 保持距离，等激怒结束 |

**推荐战术：**
1. 开场先用弓箭削弱（如果有弓术）
2. 保持中距离引诱冲锋劈
3. 弹反冲锋劈后接前冲斩+轻攻击连招
4. 注意 50% 血量时的激怒转换——他会后退并咆哮，利用这个窗口喝药
5. 激怒状态下优先闪避，弹反窗口缩短
6. 使用摔角背摔可以大量消耗架势条

> **关键提示：** 如果觉得BOSS太难，先去完成支线任务和探索圣所获取更好的装备。推荐等级 8 级、装备至少蓝色品质再挑战。

| 指标 | 推荐值 |
|------|--------|
| **角色等级** | 8-10 级 |
| **主武器** | 蓝色稀有品质以上 |

> **如果敌人过于强大**，优先完成支线任务和悬赏来获取贡献点数和更好的装备。不要急于推进主线——赫南德区域有足够的内容让你安全地提升到 10 级。
